## What This Will Become

Contexts are domain-specific contemporary awareness packages:
- Current state of a field
- Key players and developments
- Recent changes and trends
- What matters now

Unlike fu (meta-cognitive patterns), contexts are about **content knowledge** of specific domains.

- Major labs and their focuses
- Recent model developments

- Current discourse themes
