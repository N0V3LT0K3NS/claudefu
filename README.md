## Quick Start

### Use the Repo Analyzer

```bash
# Clone this repo
git clone https://github.com/N0V3LT0K3NS/claudefu.git

# Copy the skill to your global skills
cp -r claudefu/.claude/skills/repo-analyzer ~/.claude/skills/

# Now you can analyze any repo
# In Claude Code: /repo-analyzer https://github.com/some/repo
```

### Read the GROK Guide

Start with [CLAUDE_CODE_GROK_ENRICHED.md](docs/CLAUDE_CODE_GROK_ENRICHED.md) for complete operational awareness of Claude Code's capabilities.
